Oldcastle Glass Perrysburg - Accounting Department Website

Description

While working as an accountant at Oldcastle Glass Perrysburg, and on my own initiative, I learned the necessary tools and technologies to develop a departmental website that streamlined the month-end closing cycle by many hours every month.

top of page

Technologies Used

  • Java
  • PostgreSQL
  • MySQL
  • Tomcat
  • JBoss
  • JEE/J2EE
  • JSP/Servlets
  • HTML
  • Fedora Linux
  • Apache POI
  • Eclipse
  • Netbeans

top of page

Results

  • Automated repetitive and error-prone tasks, saving hours of processing time during each month-end close cycle.
  • Extracted journal entry information directly from Excel spreadsheets and uploaded it to the AS/400 for posting.
  • Reduced the time required to create the Profit and Loss Statement from 30+ minutes to a few seconds.
  • The creation of other reports experienced similar gains in processing time.
  • The creation of detailed and customized reports, otherwise unavailable from the AS/400's accounting software package.
  • Prior to leaving, I was developing a modest data warehouse for the Accounting department and plant’s use.
