Description

Coursework for the Udacity Intermediate Python Nanodegree. The course has two sections and two projects that are described below. I have paused this course for now, but my work for it can be found on GitHub via the Code link to the left.

Large Codebases with Libraries covers: PEP standards, Building Modules, Using Libraries, Python in Systems, and Python for Web. The project is a meme generator.
