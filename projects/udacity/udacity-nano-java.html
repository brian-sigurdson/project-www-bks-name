Udacity - Java Programming

Description

Coursework for the Udacity Java Programming Nanodegree. The course has three sections and projects, which are described below.

Java Fundamentals covers Java basics and object-oriented programming (OOP). The application for section 1 is a hotel reservation application. My work can be found on GitHub, via the code link found below or on the navigation bar on the left of the page.

Advanced Java Programming covers: Functional Programming, File I/O, Design Patterns, Reflection, and Concurrent Programming. The application for section 2 is a legacy web crawler. I have not completed this project yet, but my work so far is on GitHub.

Java Application Deployment covers: Running applications, Maven, Java Modules, Unit Testing, Mocking and Integration Testing. The project is focused on application deployment. I have not yet completed this application.
